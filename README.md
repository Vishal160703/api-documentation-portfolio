## Get Posts API

**Endpoint:** https://jsonplaceholder.typicode.com/posts
**Method:** GET

**Description:**
This API retrieves a list of posts from the server. The data is returned in JSON format and is commonly used to display content such as articles or blog posts in applications.

**Request:**
No request body is required.

**Parameters:**

* **Query Parameter (id):**
  Used to filter posts based on their ID. Example: `?id=1`

* **Path Parameter (post ID):**
  Used to retrieve a specific post. Example: `/posts/1`

**Headers:**

* **Content-Type:** application/json
* **Authorization:** Bearer <token>

**Response:**

* **userId**: ID of the user who created the post
* **id**: Unique identifier of the post
* **title**: Title of the post
* **body**: Content of the post

**Example Response:**

```json id="pfix1"
[
  {
    "userId": 1,
    "id": 1,
    "title": "Sample Title",
    "body": "Sample content"
  }
]
```

## Create Post API

**Endpoint:** https://jsonplaceholder.typicode.com/posts
**Method:** POST

**Description:**
This API creates a new post on the server. It accepts post details in the request body and returns the created post information.

**Request Body:**

* **title**: Title of the post
* **body**: Content of the post
* **userId**: ID of the user creating the post

**Headers:**

* **Content-Type:** application/json
* **Authorization:** Bearer <token>

**Response:**

* **title**
* **body**
* **userId**
* **id** (generated by server)

**Example Request:**

```json id="pfix2"
{
  "title": "My First Post",
  "body": "This is my content",
  "userId": 1
}
```

**Example Response:**

```json id="pfix3"
{
  "title": "My First Post",
  "body": "This is my content",
  "userId": 1,
  "id": 101
}
```
